## IMPORTANT: No Git Operations Without Permission

**NEVER** run `git add`, `git commit`, `git push`, or any other git commands that modify the repository history on this project without **explicit user permission**.

- Do NOT commit code changes
- Do NOT push to GitHub
- Do NOT force push
- Do NOT create branches

You may only edit files locally. Always ask the user before performing any git operations.
